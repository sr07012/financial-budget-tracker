# financial-budget-tracker
Developed an automated Profit &amp; Loss reporting system in Excel to monitor income, expenses, gross margin, and budget variance. The dashboard dynamically updates financial KPIs based on user inputs, enabling structured business performance analysis.
🎯 Objective

To create a structured financial tracking system that provides:

Budget vs Actual comparison

Monthly financial performance tracking

Automated Profit & Loss calculation

Visual dashboard for management insights

Tools Used

Microsoft Excel

Data Validation

Formula-based automation

Pivot-style summary calculations



Key Features

Income and Expense categorization

Automated Net Profit calculation

Gross Margin & Return metrics

Budget variance analysis

Dashboard visualization

Protected formula structure

📂 File Structure

Business Budget Template.xlsx

Income Sheet

Expense Sheet

Dashboard Summary

🚀 How It Works

Enter monthly income data.

Enter monthly expense data.

Dashboard auto-updates key metrics.

Monitor business financial health in real-time.

This system can be used by:

Small businesses

Trading companies

Manufacturing units

Financial analysts

Operations managers
